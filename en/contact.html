Louhos

Open Source Code Repository for Open Data in Finland

Contact information

You can contact the Louhos admin team via:

  1. Email: sorvi-admin - at - lists.r-forge.r-project.org
  2. IRC: !louhos@IRCnet
  3. Twitter: hashtag #louhos
  4. GitHub: https://github.com/louhos
  5. Blog: https://louhos.wordpress.com

Louhos is a community-driven collection of open source code and tools for open data in Finland. The Louhos repository is hosted at github. Contributions to facilitate the downloading, preprocessing, analysis, and visualization, written in any language, are welcome.
