Tällä sivulla ohjeita soRvi-paketin kehittäjille ja kontribuuttoreille. Tämä sivu on vielä työn alla ja ohjeita täydennetään pian.

  1. Asenna R ja soRvin vaatimat riippuvuudet
  2. Forkkaa klooni soRvin GitHub-kehitysversiosta
  3. soRvi-paketin kääntämisohjeet:

    Asenna devtools R:ssä: {% highlight r %} install.packages(devtools) library(devtools) {% endhighlight %}

    Generoi R-paketin dokumentaatio R:ssä komentosarjalla {% highlight r %} library(devtools) document("soRvi-dev") {% endhighlight %}

    Käännä paketti komentosarjalla: {% highlight r %} /myRpath/R CMD check soRvi-dev /myRpath/R CMD build soRvi-dev /myRpath/R CMD INSTALL sorvi_x.y.z.tar.gz {% endhighlight %}

    Korjaa omiin muutoksiisi liittyvät virheilmoitukset ja varoitukset

  4. Kontribuoi, mergetä masteriin, ja lähetä ylläpidolle pull-request

Lähdekoodi

Kehitysversion lähdekoodi löytyy Louhoksen github-sivulta.

Koodi on julkaistu avoimella FreeBSD-lisenssillä, joka sallii lähdekoodin vapaan käytön, muokkaamisen ja levittämisen, kunhan samat oikeudet jatkokäytölle taataan uusissakin tuotoksissa.
