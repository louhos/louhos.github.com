sorvi on yhteiskunnan avointen tietovarantojen keskitettyyn hakuun, putsaamiseen, analysointiin, yhdistelyyn ja visualisointiin suunnattu avoimen lähdekoodin hanke, joka keskittyy erityisesti suomalaiseen avoimeen dataan. Avoimen lähdekoodin R-kielelle toteutetun kirjaston välineet helpottavat tiedonlouhinta-algoritmien soveltamista julkisiin tietolähteisiin ammattimaisessa laskentaympäristössä. Valmiiden hakurutiinien ansiosta monipuoliset aineistot ja tehokkaat analyysimenetelmät ovat välittömästi yhdistettävissä ilman hankalia esikäsittelyvaiheita. soRvi-kirjasto toimii mm. Linux-, Mac- ja Windows-ympäristöissä. Alla tiiviit asennusohjeet. Pyydämme raportoimaan havaituista puutteista (yhteystiedot).

Asennusohjeet

  1. Asenna R-laskentaympäristö
    Tuemme uusinta R-versiota (R-2.15.1); saattaa toimia myös aikaisemmilla. Graafisista R-käyttöliittymistä suosittelemme ohjelmistoa RStudio.
  2. Asenna riippuvuudet
    • Windows: Asenna RTools.
    • Mac: Asenna XCode. Lisäksi voit asentaa joissakin esimerkeissä tarvittavan ohjelmiston rgdal (rgdal-asennusohjeet; rgdal-asennusohjeet2).
    • Linux: Asenna XML, GEOS, PROJ.4 ja CURL. Lisäksi voit asentaa joissakin esimerkeissä tarvittavat (GDAL ja freeglut). Asentamisen sujuvoittamiseksi voit käyttää valmista asennusskriptiä (Debian / Ubuntu / Fedora / OpenSuse).
  3. Asenna soRvi
    Kun vaaditut riippuvuudet on asennettu (ks. yllä), soRvi-paketin voi asentaa suoraan R:stä käsin. Käynnistä R ja asenna devtools:

    {% highlight r %} install.packages("devtools") {% endhighlight %}

    Nyt soRvin asentaminen onnistuu R:stä käsin komennolla: {% highlight r %} library(devtools) install_github(repo = "soRvi", username = "louhos") {% endhighlight %}

  4. Kokeile asennuksen onnistumista

    Lataa soRvi käyttöön R:ssä komennolla {% highlight r %} library("sorvi") {% endhighlight %}

    Jos asennus onnistui, ruudulle tulostuu tietoa paketista. Seuraavaksi voit kokeilla vaikkapa seuraavaa esimerkkiä: {% highlight r %} source("http://sorvi.r-forge.r-project.org/examples/helloworld.R") {% endhighlight %}

Yhteisö ja tuki

Kaikki palaute asennukseen ja käyttöön liittyen on erittäin tervetullutta; kehittäjäyhteisöltä saaa tukea kirjaston asentamiseen, käyttöön ja kehittämiseen (yhteystiedot). Projektin pääkehittäjiä ovat Leo Lahti, Juuso Parkkinen ja Joona Lehtomäki. Uudet tekijät ja kontribuutiot ovat tervetulleita. Osallistumismahdollisuuksia on listattu wiki-sivulle.
