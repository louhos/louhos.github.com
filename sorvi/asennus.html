soRvi on yhteiskunnan avointen tietovarantojen keskitettyyn hakuun, putsaamiseen, analysointiin, yhdistelyyn ja visualisointiin suunnattu avoimen lähdekoodin hanke, joka keskittyy erityisesti suomalaiseen avoimeen dataan. Avoimen lähdekoodin R-kielelle toteutetun kirjaston välineet helpottavat tiedonlouhinta-algoritmien soveltamista julkisiin tietolähteisiin ammattimaisessa laskentaympäristössä. Valmiiden hakurutiinien ansiosta monipuoliset aineistot ja tehokkaat analyysimenetelmät ovat välittömästi yhdistettävissä ilman hankalia esikäsittelyvaiheita. soRvi-paketti toimii mm. Linux-, Mac- ja Windows-ympäristöissä. Alla tiivis ohjeistus kehitysversion käyttöönotosta. Pyydämme raportoimaan havaituista puutteista osoitteeseen sorvi-commits at lists.r-forge.r-project.org

  1. Asenna R-laskentaympäristö
    Tuemme uusinta versiota (R-2.14.0), joskin saattaa toimia myös aikaisemmilla. Graafisista R-käyttöliittymistä suosittelemme ohjelmistoa RStudio.
  2. Asenna riippuvuudet
    • Windows: Suurin osa riippuvuuksista asentuu tarvittaessa automaattisesti. Asenna lisäksi PERL. Jos haluat asentaa soRvi:n kehitysversion, asenna myös RTools.
    • Mac: Asenna XCode. Lisäksi voit asentaa joissakin esimerkeissä tarvittavan ohjelmiston rgdal (rgdal-asennusohjeet; rgdal-asennusohjeet2).
    • Linux: Aja komentoriviltä asennuskripti. Tämä asentaa ohjelmistot XML, GEOS, PROJ.4 ja CURL, sekä joissakin esimerkeissä tarvittavia paketteja (GDAL ja freeglut). Jos asennusskriptin ajamisessa tulee ongelmia, voit kokeilla mainittujen pakettien manuaalista asentamista.
  3. Asenna soRvi

    soRvi:n voi asentaa joko julkaisu- tai kehitysversion. Julkaisuversio on vaakaampi ja yleensä saatavilla kaikille alustoille (Win, Mac ja Linux). Kehitysversio sisältää uusimmat työkalut ja ajantasaisimmat aineistot, mutta on epävakaampi ja koodi saattaa muuttua paljonkin ennen julkaisua. soRvi:n kehitystyöstä kiinnostuneille kehitysversio on suositeltava versio.

    Julkaisuversio

    Käynnistä R ja aja sorvin asennusskripti R:stä käsin komennolla {% highlight r %} source("http://sorvi.r-forge.r-project.org/examples/sorvi.installation.R") {% endhighlight %}

    Kehitysversio

    Asenna ensin paketti devtools: {% highlight r %} install.packages(devtools) library(devtools) {% endhighlight %}

    Asenna soRvi:n kehitysversio suoraan GitHubista. {% highlight r %} install_github(repo = "soRvi-dev", username = "louhos", type = "source") {% endhighlight %}

  4. Kokeile asennuksen onnistumista

    Lataa soRvi käyttöön R:ssä komennolla {% highlight r %} library("sorvi") {% endhighlight %}

    Jos asennus onnistui, ruudulle tulostuu tietoa paketista. Seuraavaksi voitkokeilla vaikkapa seuraavaa esimerkkiä: {% highlight r %} source("http://sorvi.r-forge.r-project.org/examples/helloworld.R") {% endhighlight %}

Lisää esimerkkejä paketin käytöstä Suomi-datan penkomiseen löydät manuaalista, Louhos-blogista ja koodiarkistosta. Palaute asennukseen ja käyttöön liittyen on tervetullutta; tarjoamme tukea mahdollisuuksien mukaan.

Lähdekoodi

Kehitysversion lähdekoodi löytyy Louhoksen github-sivulta.

Koodi on julkaistu avoimella FreeBSD-lisenssillä, joka sallii lähdekoodin vapaan käytön, muokkaamisen ja levittämisen, kunhan samat oikeudet jatkokäytölle taataan uusissakin tuotoksissa.
